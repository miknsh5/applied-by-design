Project Overview

This project began as a proof of concept into building data-driven web applications to support our aviation clients. Airlines are beginning to make better operational and strategic investment decisions based on the data that's driving their business.


Key to unlocking the value in data is the ability to interact with and interrogate information in ways that allow users to derive meaning. This project explores the use of rapid application development and data visualization solutions to build a platform for airline data analytics.

Technology Selection

We developed this application primarily in AngularJS, driven by Node.js on the backend and a MongoDB database.
